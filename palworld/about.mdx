Our Palworld server is a mostly default server offering a laid back and tightly knit community.

We are focused around building a long-term community, with reduced XP and no guild resets. The server run in a seasonal format, with breaks between seasons.

## General Information

<Note>
  **This server is whitelisted!**

  In order to join, you'll need to apply first at [https://trixtail.com/whitelist](https://trixtail.com/whitelist).

  Before joining, we strongly recommend you read the [community guidelines](/palworld/community-guidelines).
</Note>

## Custom World Settings

Any setting not specified here is on its default value for the **Normal** difficulty.

- **Structure Deterioration Rate:** Disabled
- **Max Bases Per Guild:** 6
- **Max Work Pals At Base:** 25
- **Global Palbox Import:** Disabled
- **XP Rate:** 0.8x
- **Day Length:** 23 minutes
- **Night Length:** 7 minutes

If you would like any further adjustments to the World Settings, make a suggestion on our Discord!

<Callout icon="fire" color="#FFA500">
  **<u>Challenge Mode: Only Breed Known Pals</u>**

  Returning from Season 1, we are encouraging users to undertake this challenge to change up their gameplay experience!

  If you opt-in to this challenge, you cannot breed Pals to obtain a Pal that you haven't yet caught in the wild at least once. This is to maintain the prestige and value of Pals native to high level areas.

  Following this rule is strictly **optional.**
</Callout>
