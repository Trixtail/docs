Our Palworld server is a mostly default server offering a laid back and tightly knit community.

We are focused around building a long-term community, with reduced XP and no guild resets. The server run in a seasonal format, with breaks between seasons.

## Custom World Settings

Any setting not specified here is on its default value for the **Normal** difficulty.

- **Structure Deterioration Rate:** Disabled
- **Max Bases Per Guild:** 6
- **Max Work Pals At Base:** 30
- **Base Raids:** Disabled
- **XP Rate:** Varies, starts at 0.5x (See Sliding XP Scale)
- **Day Length:** 25 minutes
- **Night Length:** 15 minutes

If you would like any further adjustments to the World Settings, make a suggestion on our Discord!

## Sliding XP Scale

We are trialing an experimental sliding XP scale where the global XP multiplier slowly increases over the season, giving less active players an opportunity to catch up.

The current XP modifier is determined based on the median level of the playerbase, making levelling slower if you're ahead of the curve and quicker if you're behind!

As a tradeoff, this system makes general levelling slower in the earlygame and faster in the lategame.

## XP Boundaries

Boundaries are determined based on the level of each Tower Boss.

| Median Player Level | XP Multiplier |
| --- | --- |
| 0-24 | 0.5x |
| 25-39 | 0.7x |
| 40-49 | 1.0x |
| 50-54 | 1.2x |
| 55-59 | 1.5x |
| 60\+ | 2.0x |

These boundaries are subject to review.
